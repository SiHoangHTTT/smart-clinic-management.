# User Stories - Smart Clinic Management System

## Doctor
- As a doctor, I can view my patient appointments so that I can prepare for consultations.
- As a doctor, I can update prescriptions for a patient to ensure correct medication is given.
- As a doctor, I can see patient medical history to make informed treatment decisions.
- As a doctor, I can mark appointments as completed to update patient records.
- As a doctor, I can set my available times so that patients can book appointments accordingly.

## Patient
- As a patient, I can register and create an account to use the clinic system.
- As a patient, I can book an appointment with a doctor for a specific date and time.
- As a patient, I can view my upcoming appointments to stay informed.
- As a patient, I can cancel or reschedule an appointment to manage my schedule.
- As a patient, I can view my prescriptions to follow the doctorâ€™s instructions.

## Admin
- As an admin, I can add new doctors to the system to manage clinic staff.
- As an admin, I can remove or update doctor information to keep records accurate.
- As an admin, I can assign roles (Doctor/Patient) to users to control access.
- As an admin, I can view all appointments to monitor clinic operations.
- As an admin, I can generate reports on doctor performance and patient visits.
